# Curso em Vídeo - HTML5 e CSS3

Este repositório contém os arquivos dos exercícios do curso de HTML5 e CSS3 que estou fazendo no site Curso em Vídeo. O curso é ministrado pelo professor Gustavo Guanabara e é voltado para iniciantes que desejam aprender a criar páginas web utilizando as tecnologias HTML5 e CSS3.
